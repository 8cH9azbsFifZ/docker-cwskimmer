Run CW Skimmer under docker.

# How to use
+ `make build`
+ `make run`
+ `open http://127.0.0.1:8080`

# Configuration

## Hermes DLL
+ Setup IP: HermesIntf_.dll e.g. HermesIntf_192.168.111.222.dll
